# Cyberk Website — Content Plan

> Tài liệu hướng dẫn team marketing chuẩn bị nội dung cho website cyberk.io.
> Bao gồm: cấu trúc trang, SEO keywords, nội dung cần viết, và cross-linking.

---

## 1. Tổng quan Cấu trúc Website

Website cyberk.io gồm **4 nhóm trang chính** + các trang phụ trợ:

```
cyberk.io

│   └── /how-we-build

├── 📊 Case Studies (Bằng chứng — CYBERK ĐÃ LÀM GÌ)
```

### Vai trò từng nhóm trang

| Nhóm | Vai trò | Khách hàng đến đây khi... |
|---|---|---|
| **Services** | Bán hàng, conversion | Muốn biết "mua gì, bao nhiêu tiền" |
| **Capabilities** | Trust + SEO traffic | Search "smart contract development" trên Google |
| **How We Build** | Trust + differentiation | Muốn biết "Cyberk làm việc như thế nào" |
| **Case Studies** | Proof + SEO | Muốn thấy "Cyberk đã làm gì thành công" |

## 2. Mối quan hệ: Services ↔ Capabilities

Đây là điểm quan trọng nhất cần hiểu:

- **Services** = "Cái khách hàng MUA" — gói giải pháp có giá cả, deliverable
- **Capabilities** = "Lý do khách hàng TIN" — năng lực kỹ thuật chuyên sâu

```
Visitor tìm "smart contract development company" trên Google
  → Google → /capabilities/smart-contract-development
    → Đọc, thấy expertise + case studies → TIN TƯỞNG
      → CTA: "Ready to build? → /services/product-build"
        → Thấy giá + scope + process → MUA

Visitor tìm "build web3 product"  
  → Google → /services/product-build
    → Thấy giá, OK nhưng Cyberk có đủ giỏi không?
      → Link → /capabilities/blockchain-development
        → Thấy expertise + case studies → TIN → QUAY LẠI MUA
```

**Capabilities kéo traffic. Services convert thành tiền.**

## 4. Chi tiết: Capabilities Pages (NỘI DUNG MỚI CẦN VIẾT)

> ⚠️ Đây là nội dung CHƯA CÓ, cần team marketing viết mới.
> Mỗi trang cần tối thiểu **800–1200 từ** tiếng Anh để rank tốt trên Google.

### Template chung cho mỗi Capability page

```
1. Hero Section
   - Headline (có primary keyword)
   - Subheadline (1-2 câu mô tả)
   - CTA button

2. Overview (2-3 paragraphs)
   - Capability này là gì?
   - Tại sao quan trọng?
   - Cyberk làm gì trong lĩnh vực này?

3. What We Build (bulleted list)
   - Liệt kê cụ thể các loại sản phẩm/giải pháp

4. Tech Stack
   - Badges/logos công nghệ sử dụng

5. Our Approach (3-4 bước)
   - Quy trình Cyberk tiếp cận capability này

6. Case Studies (2-3 dự án)
   - Tên dự án + mô tả ngắn + kết quả
   - Link sang trang case study chi tiết

7. Why Cyberk? (3-4 bullet points)
   - Differentiators: Framework, Solo Dev, CTO mindset...

8. CTA Section
   - Link sang Service pages: Product Design / Product Build
   - Contact form hoặc booking link
```
